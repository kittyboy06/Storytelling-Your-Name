04 The Disaster Logic: Event Propagation

CASCADE DELETE

When the Comet impact Event (T=0) occurs, it triggers a cascade delete across all Town-related entities. All Foreign Keys linked to 'Itomori' are set to NULL or Deleted.

ATOMICITY

The disaster is an Atomic Transaction; either the entire town survives or the entire town record is lost. There is no partial state allowed in the database.
